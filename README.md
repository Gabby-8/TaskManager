
# Task Manager

Task Manager is a simple full-stack web application built as a practice project to improve my skills in React and C# (.NET Web API).


## Appendix

The goal of this project is to understand how a frontend application communicates with a backend API, and how basic CRUD operations are implemented in a full-stack setup.


## Tech Stack

**Frontend:** 
- React (Vite)
- JavaScript
- HTML & CSS (no UI frameworks)

**Backend:**
- C# (.NET Web API)
- ASP.NET Core
- RESTful API architecture
- Swagger for API testing


## Features

- View list of tasks
- Add new tasks
- Edit existing tasks
- Delete tasks
- REST API communication between React and C#
- Simple and clean UI using plain CSS


## Purpose

**This project was created for learning and practice purposes to:**

- Strengthen understanding of React fundamentals (state, hooks, components)
- Learn how to build and consume a C# Web API

- Practice CRUD operations

## Run Locally

**Clone the project**

```bash
  git clone https://github.com/Gabby-8/TaskManager
```

**Navigate to backend API**

```bash
  cd backend/TaskApi
```

**Run backend**

```bash
  dotnet run
```

**Navigate to frontend**

```bash
  cd frontend/task-ui
```

**Install dependencies**

```bash
  npm install
```

**Run frontend**

```bash
  npm run dev
```
